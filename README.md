# Neural_Network_From_Scratch
Neural Network implementation from scratch on Dummy Data and MNIST using NumPy.
## MNIST From Scratch

This repository contains a neural network built from scratch using NumPy
to classify handwritten digits from the MNIST dataset.

## Notebooks
- **NN_Scratch_ Understanding_ dummydata.ipynb**  
  Dummy Data loading, normalization, and one-hot encoding

- **NN_Scratch_ Understanding_MNIST.ipynb**  
  Forward propagation, predictions, and correct vs incorrect visualization

## Key Concepts
- Forward propagation
- Back Propagation
- Dense Layer
- ReLu Activation and Softmax
- One-hot encoding & Loss
- Neural networks without frameworks
